# AI-GAME

## Relic Runner (terminal game)

A simple dungeon game paced to last around **30 minutes** for a typical human player.

### How to run

```bash
python3 game.py
```

### Objective

- Clear 5 floors.
- On each floor, collect 3 relics.
- After collecting all relics, reach the exit tile to move on.

### Commands

- `north`, `south`, `east`, `west` — move
- `look` — show explored mini-map
- `stats` — display health, attack, and progress
- `heal` — use a healing potion
- `help` — list commands
- `quit` — leave the game
